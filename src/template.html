H.A.H.N.S

Hardware, Advisories, Highlights, & Navigation Specialist

A no-install helper that reads the repair procedure you're already looking at and lists the torque specs, one-time-use parts, fluids, special tools, and warnings — and keeps nothing.

Version __BUILD__ — if the panel on ELSA shows a different version, re-drag the bookmark.

Your data stays put. This tool runs entirely inside your own browser. It only reads text that ELSA has already shown you, it never saves a file, and it never sends anything over the internet. Close the panel and it's gone.

1Add the wrench to your bookmarks bar

Make sure your bookmarks bar is showing (Ctrl+Shift+B in Chrome/Edge), then drag this button up onto it:

H.A.H.N.S

Dragging it = saving a bookmark. Nothing installs on your computer.

Can't drag it? Add it by hand instead
  1. Right-click the bookmarks bar → Add page…
  2. Name it H.A.H.N.S
  3. In the URL/Address box, paste the code from the file bookmarklet.txt (in this folder), then Save.

2Use it on a job

  1. Log into vw-now.com and open a repair procedure in ELSA as usual.
  2. Click the H.A.H.N.S bookmark.
  3. The wrench panel appears with your specs sorted. Hit Re-scan after you navigate to a new page, or the ✕ to close.
If the panel comes up empty on a real procedure, the spec text is probably inside an embedded frame from a different web address, which a bookmark can't reach. Use the paste box below as a fallback — copy the procedure text and drop it in.

3Try it now (no login needed)
